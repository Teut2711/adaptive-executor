Running Examples
----------------

1. Install Adaptive Executor with all features:

.. code-block:: bash

   pip install adaptive-executor[standard]

2. Run any example:

.. code-block:: bash

   python examples/basic_usage.py

For more examples and details, see the :doc:`examples directory <../examples/>`.
